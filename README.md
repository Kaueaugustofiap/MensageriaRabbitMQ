<h1>Projeto de Mensageria com RabbitMQ</h1> 

<h2>Basicamente o projeto contém:</h2>

* Um producer, que deve enviar os dados para um Broker (RabbitMQ).
*  Um consumer, que deve ler esses dados e processá-los.
