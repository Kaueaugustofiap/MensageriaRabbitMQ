<h1>Projeto de Mensageria com RabbitMQ</h1> 

<h2>Basicamente o projeto contém:</h2>

* Um producer, que envia os dados para um Broker (RabbitMQ).
*  Um consumer, que lê esses dados e processa os mesmos.
